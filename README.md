# Laboratorio 04 - Ajedrez en Python

## Requisitos

- Python 3.x
- Pygame
- NO python 3.14
---

## Crear y activar entorno virtual

```bash
# Crear entorno virtual
python -m venv daw

# Activar entorno virtual

# Windows
daw\Scripts\activate

# Linux/macOS
source daw/bin/activate
```

---

## Instalar dependencias

```bash
pip install -r requirements.txt
```
